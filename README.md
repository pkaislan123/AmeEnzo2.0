# AmeEnzo2.0


O código a seguir foi usado numa cadeira de rodas feita para um garotinho que possui ELA para ajudar na sua locomoção. A cadeira foi adaptada com motores de parabrisa em cada roda da cadeira, controladas por uma ponte H. O arduino tinha como papel ser a placa-mãe, um joystick analogica era usado para dirigir a cadeira.
